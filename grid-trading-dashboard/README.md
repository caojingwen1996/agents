# 网格交易收益仪表盘

这是一个只在本机运行的单只 A 股网格交易复盘工具。它固定读取 `data/交易记录.xlsx`，自动获取日线行情，并展示价格、持仓成本、各网格买卖点、XIRR、仓位和网格收益。

## 使用方法

1. 打开 `data/交易记录.xlsx`，修改“配置”和“交易流水”。
2. 双击 `启动工具.bat`。首次运行会安装所需组件，时间稍长。
3. 浏览器会自动打开 `http://127.0.0.1:8765`。
4. Excel 修改并保存后，点击页面右上角“刷新数据”。
5. 使用结束后，关闭启动工具的黑色窗口即可停止服务。

交易流水和缓存都保存在本机。本工具不会上传交易数据，也不会连接券商或执行交易。

## 配置工作表

| 字段 | 填写方式 |
| --- | --- |
| 股票代码 | 六位 A 股代码，例如 `000001` |
| 图表标题 | 主图标题，例如“网格波段 提款足迹” |
| 行情复权 | 固定填写“不复权” |
| 策略预算 | 该策略可使用的总金额，用于计算单品仓位 |

## 交易流水工作表

| 字段 | 填写方式 |
| --- | --- |
| 日期 | `YYYY-MM-DD` |
| 类型 | 买入、卖出或分红 |
| 网格编号 | 买卖必填；同一格的买入和卖出填写相同编号，例如 `G3` |
| 数量 | 买卖股数；分红可填 0 |
| 成交价 | 买卖实际成交价；分红可填 0 |
| 手续费 | 该笔实际手续费，没有则填 0 |
| 印花税 | 该笔实际印花税，没有则填 0 |
| 现金金额 | 分红实际入账金额；买卖填 0 |
| 备注 | 可选说明 |

同一网格的卖出按先进先出与买入配对。卖出超过该网格可卖数量时，页面会指出具体 Excel 行号并停止本次刷新，不会自动修改数据。

## 行情缓存与错误

行情缓存在 `data/cache/`。网络不可用但缓存足够时，页面继续显示最近结果，并标明行情截止日期。如果 Excel 缺列、格式错误、网格编号缺失或发生超额卖出，页面顶部会显示具体原因，同时保留最近一次成功结果。
