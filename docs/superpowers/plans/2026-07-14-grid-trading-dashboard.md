# Grid Trading Dashboard Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a Windows-local dashboard that reads `data/交易记录.xlsx`, fetches and caches one A-share's daily prices, calculates grid-trading returns, and renders the approved black ECharts dashboard.

**Architecture:** A small Flask application delegates Excel parsing, grid/portfolio calculations, and market-data caching to focused Python modules. The web layer exposes a dashboard page and refresh endpoint; the browser receives a ready-to-render JSON view model and uses a locally vendored ECharts asset so cached reports still render without internet access.

**Tech Stack:** Python 3.11, Flask, pandas, openpyxl, AKShare, PyXIRR, ECharts, pytest

---
